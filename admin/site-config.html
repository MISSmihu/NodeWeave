邮箱验证强制

🪪 实名制

实名认证模式
关闭：不要求实名 · 可选：引导但不强制 · 强制：必须实名才能发帖/评论
手机验证
用户绑定手机号的验证方式。关闭后无需绑定手机。

🔗 OAuth 三方登录

GitHub 登录
需配置 GITHUB_CLIENT_ID / GITHUB_CLIENT_SECRET
QQ 登录
需配置 QQ_APP_ID / QQ_CLIENT_ID / QQ_CLIENT_SECRET
Google 登录
需配置 GOOGLE_CLIENT_ID / GOOGLE_CLIENT_SECRET（注意数据出境合规）
GitHub 老号免邀请码
GitHub 账号注册年限超过阈值可跳过邀请码
GitHub 年限阈值（天）
GitHub 账号创建超过此天数可免邀请码
