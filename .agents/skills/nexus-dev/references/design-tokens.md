# 设计 Tokens（配色 / 字体 / 间距 / 发光）

NEXUS 赛博朋克风的视觉 DNA，全部定义在 `cyber-forum/style.css` 的 `:root`。**所有页面必须引用这些变量，禁止硬编码颜色/字体。**

## 配色（核心霓虹色板）

| 变量 | 值 | 用途 |
|------|-----|------|
| `--bg` | `#0a0a0f` | 页面主背景（近黑） |
| `--bg-2` | `#0f0f1a` | 卡片渐变末端、次级背景 |
| `--surface` | `#12121e` | 卡片/容器背景（渐变起点） |
| `--surface-2` | `#1a1a2a` | 嵌套容器、chip、滚动条 |
| `--border` | `rgba(0,240,255,.12)` | 主边框（带青色调） |
| `--border-2` | `rgba(255,255,255,.06)` | 次级分隔线 |
| `--cyan` | `#00f0ff` | **主色**：链接、激活态、主按钮、标题高亮 |
| `--magenta` | `#ff003c` | **强调色**：警告、置顶、危险操作 |
| `--purple` | `#9d00ff` | 装饰渐变、第三排名色 |
| `--green` | `#7fff00` | 成功、在线、正向趋势、终端绿点缀 |
| `--text` | `#e8e8ff` | 正文主色 |
| `--text-dim` | `#8a8aa8` | 次级文字、描述 |
| `--text-mute` | `#5a5a78` | 元信息、时间戳、占位符 |

**配色语义约定**（什么场景用什么色）：
- 主要操作 / 激活 / 信息 → `--cyan`
- 危险 / 删除 / 报错 / 置顶 → `--magenta`
- 成功 / 在线 / 正向数据 → `--green`
- 渐变装饰（标题、光晕、排名）→ `--cyan` ↔ `--purple` 或 `--magenta` ↔ `--purple`

## 发光阴影（赛博风的灵魂）

```css
--glow-cyan:    0 0 20px rgba(0,240,255,.45), 0 0 40px rgba(0,240,255,.15);
--glow-magenta: 0 0 20px rgba(255,0,60,.45),  0 0 40px rgba(255,0,60,.15);
```

**用法**：在 hover、激活、强调元素上叠加 `box-shadow` 或 `text-shadow`。一个页面不能"全发光"（会糊成一团），发光是**点缀**，用在最该被注意的 1-2 个元素上。

新增紫色/绿色发光时按同样比例定义 `--glow-purple` / `--glow-green`。

## 字体（只用三套）

| 变量 | 字体 | 用途 |
|------|------|------|
| `--font-display` | `'Orbitron', 'Noto Sans SC'` | 标题、Logo、数字统计、按钮文字（科技感） |
| `--font-mono` | `'JetBrains Mono'` | 代码、标签 chip、元信息、时间戳、数据 |
| `--font-body` | `'Noto Sans SC', system-ui` | 中文正文、描述、卡片摘要 |

**引入**：已在 `index.html` 通过 Google Fonts CDN 引入。新页面**复制同样的 `<link>`** 到 `<head>`，不要加新字体。

**字体使用口诀**：中文正文用 body，英文/数字/代码用 mono，标题用 display。混排时（如"NEXUS 系统"）中英分别用对应字体族。

## 间距与圆角

项目用 **8px 网格**，没有专门的变量，直接用值：

| 用途 | 值 |
|------|-----|
| 紧凑内边距 | `8px` / `12px` |
| 卡片内边距 | `18px` / `22px 24px` |
| 区块间距 | `18px`（卡片间）/ `32px`（布局列间） |
| 大区块纵向间距 | `70px` - `90px`（Hero 等） |
| 圆角 | 卡片 `8px`、按钮 `4px`、chip/标签 `3px`、圆形头像 `50%` |

**赛博风偏好小圆角**（4-8px），不用大圆角（16px+）。按钮可用 `clip-path` 做斜切角增加机械感（见 `.btn-primary`）。

## 响应式断点

```css
@media (max-width: 1024px) { /* 平板：侧边栏横向铺开 */ }
@media (max-width: 768px)  { /* 移动端：隐藏导航次要项、单列布局 */ }
```

移动端至少测一遍：导航折叠、卡片单列、Hero 字号缩小（用 `clamp()`）。

## 文案口吻

- 简洁、略带极客感
- 中英混排自然（"SYSTEM ONLINE · 已连接 1247 个节点"）
- 状态用英文标签 + 中文说明
- 数字用 `toLocaleString()` 千分位
